DOM (Document Object Model) Manipulation

The DOM stands for Document Object Model, It is essentially a programming interface for web/HTML documents.

Document Properties/Elements

We can access DOM elements directly, such as the body

console.log(document.body);

We can also access elements such as links and images. Let's add a link to the HTML

href="https://siltawi.com"

We can get all of the links with

console.log(document.links);

Document Selection Methods

Getting elements using direct properties is not usually the way you want to do this. There are special methods available that allow us to directly select elements easily

href="https://siltawi.com" id='pageLink'
console.log(document.getElementById('pageLink'));

Now let's say we want to add some text into the div element. We can easily do that using the innerHTML property.

document.getElementById('pageLink').innerHTML = 'Siltawi Training is Awesome';

The DOM was created to be independent of any particular language. Yes, JavaScript is definitely the most common language to do this stuff with, but implementations of the DOM can be built for any language, as this Python example demonstrates: 

    # Python DOM example
    import xml.dom.minidom as m
    doc = m.parse(r"C:\python_ex1.xml")
    doc.nodeName # DOM property of document object
    p_list = doc.getElementsByTagName("para")

We can get element ById(), ByClassName(), or ByTagName() method to get a single element. Take a look an example: 

let title = document.getElementById(‘title-id’);

let elements = document.getElementsByClassName(‘list-elements’);

let paragraphsEl = document.getElementsByTagName(‘p’);

Changing HTML Elements

With HTML DOM we can change the content and style of an HTML element by changing its attribute. We use innerHTML attribute for changing the content of an HTML element.

document.getElementById("#title").innerHTML = 'Welcome to Siltawi!';
document.getElementsByTagName("div").innerHTML = 'h1 Welcome to Siltawi! /h1';

Adding and deleting elements

We can change almost everything about the DOM data structure. By changing parent-child relationships the shape of the document tree can be modified. Nodes have a remove method for removing them from their current parent node. If we want to add a child node to an element node, we can use appendChild, which puts it at the end of the list of children, or insertBefore.

let p = document.createElement(‘p’);
let newContent = document.createTextNode("Welcome to Siltawi!"); 
p.appendChild(newContent);
document.body.insertBefore(p, currentP);

element.parentNode.removeChild(element);

  • mouse click
  • page load

  • input field change
