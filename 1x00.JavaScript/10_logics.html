Conditions

  1. If Statements

    In any programming language, there will be times where you need to execute certain blocks of code and commands if a certain condition is true or false. 

        if (condition) {
        // code to execute if condition is true
        }

    //if else Statements
    const x = 10;

    if (x > 5) {
    console.log(`${x} is greater than 5`);
    } else {
    console.log(`${x} is less than or equal to 5`);
    }

  2. Else If Statements

    There may be cases where you want to execute different code blocks depending on multiple conditions. 

        const d = new Date();
    const hour = d.getHours();
    if (hour < 12) {
        console.log('Good Morning!');
    } else if (hour < 18) {
        console.log('Good Afternoon!');
    } else {
        console.log('Good Night!');
    }

  3. Nesting

    There may be cases where you need to have if statements inside of if statements. 

        if (hour < 12) {
        console.log('Good Morning!');
        
        if (hour === 6) {
            console.log('Wake up!');
        }
        } else if (hour < 18) {
        console.log('Good Afternoon!');
        } else {
        console.log('Good Night!');
        
        if (hour >= 20) {
            console.log('zzzzzzzzzzz!');
        }
    }

  4. Multiple Conditions

    We can test for multiple conditions in the same if statement by using the && (AND) and the || (OR) logical operators.

        if (hour >= 7 && hour < 15) {
        console.log('It is work time!');
        }
        
    if (hour === 6 || hour === 20) {
    console.log('Brush your teeth!');
    }

  5. Switches

    If you find yourself with a lot of else-if statements and you are testing a single value, you may want to consider using a switch statement.

        switch (expression) {
        case value1:
            statement1;
            break;
        case value2:
            statement2;
            break;
        default:
            statement3;
        }

  6. Ternary Operator

    There is a third way to write conditional logic. This is called the ternary operator. It is basically a shorthand, one line if-else statement. The actual operator is just a question mark. 

        condition ? true : false;

    const auth = true;

    const redirect = auth
    ? (alert('Welcome To The Dashboard'), '/dashboard')
    : (alert('Access Denied'), '/login');

    console.log(redirect);

    auth && console.log('Welcome to the dashboard');
