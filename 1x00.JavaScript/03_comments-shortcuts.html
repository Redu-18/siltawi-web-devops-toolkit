Comments

There are two types of comments :

  1. Single Line
    2. // This is a single line of code
  2. Multiple Line
    3. /* This is a multiple
    line of code */

Shortcuts

These work for VS Code, but many of them are universal and will work in most text editors.

  • shift + up/down - Highlight lines of code up and down
  • shift + right/left - Highlight code right and left
  • ctrl + right/left arrow - move cursor to beginning/end of line
  • ctrl + up/down arrow - move cursor to beginning/end of file
  • alt + up/down arrow - move line up/down
  • shift + alt + up/down arrow - copy line up/down
  • ctrl + / - Comment out a line of code
  • ctrl + shift + / - Comment out multiple lines of code
  • ctrl + d - Select the next instance of the selected word
  • ctrl + shift + l - Select all instances of the selected word
  • ctrl + shift + f - Search for a string in the project
  • ctrl + shift + o - Search for a file in the project
