Array

Arrays are a special type of object in JavaScript.

const num = [15, 25, 60, 23, 1, 301]; 

        const str = ['chair', 'table', 'white board'];
  • Array Length
    num.length
  • Manipulating
    Assigning: 
    str[2] = 'laptop'

    Adding to Last: 
    str.push('table')

    Adding to First: 
    str.unshift('speaker')

    Removing from Last: 
    str.pop()

    Removing from First: 
    str.shift()

    Flip/Reverse: 
    num.reverse()

    Concatenate: 
    num.concat(str)
