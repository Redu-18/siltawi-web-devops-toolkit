Math

JavaScript has a built-in object called Math. This object has a bunch of methods that have to do with mathematical operations.

  • Math.sqrt()

    Returns the square root of a number
    Math.sqrt(25);

  • Math.round()

    Returns the rounded value of a number
    Math.round(3.5);

  • Math.floor()

    Returns the largest integer less than or equal to a number (rounds down)
    Math.floor(3.5);

  • Math.ceil()

    Returns the smallest integer greater than or equal to a number (rounds up)
    Math.ceil(3.3);

  • Math.abs()

    Returns the absolute value of a number
    Math.abs(-6);

  • Math.pow()

    Returns the value of a number raised to the specified power
    Math.pow(2, 3);

  • Math.min() & Math.max()

    Returns the smallest and largest of numbers listed respectively
    Math.min(2, 3, 1, 6);
    Math.max(2, 3, 1, 6);

  • Math.random()

    Returns a random number between 0 and 1
    Math.random();
